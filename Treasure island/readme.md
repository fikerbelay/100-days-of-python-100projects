#  Treasure Island

##  Project Description
Treasure Island is a text-based adventure game where the player makes a series of choices in search of hidden treasure. Every decision affects the outcome, and only the correct path leads to victory.

This project introduces nested conditional statements and demonstrates how user input can control the flow of a program.

##  How It Works
1. The game welcomes the player to Treasure Island.
2. The player is presented with different choices throughout the adventure.
3. Each decision determines the next part of the story.
4. Incorrect choices result in a "Game Over" message.
5. Choosing the correct sequence of actions leads to finding the treasure and winning the game.

##  Example

```
Welcome to Treasure Island.
Your mission is to find the treasure.

You're at a crossroad. Where do you want to go?
Type "left" or "right"
> left

You've come to a lake.
Type "wait" to wait for a boat or "swim" to swim across.
> wait

You arrive at an island with three doors.
One red, one yellow, and one blue.
Which colour do you choose?
> yellow

 You found the treasure! You Win!
```

- Conditional statements (`if`, `elif`, `else`)

- String comparison
- Program flow control
- Creating interactive console applications
