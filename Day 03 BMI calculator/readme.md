#  BMI Calculator

##  Project Description
The BMI Calculator is a simple Python program that calculates a user's Body Mass Index (BMI) using their height and weight.

After calculating the BMI, the program classifies the user into one of the following categories:
- Underweight
- Normal weight
- Overweight

This project introduces decision-making in Python using conditional statements.

##  How It Works
1. The program welcomes the user.
2. It asks for the user's height.
3. It asks for the user's weight.
4. It calculates the BMI using the formula:

   BMI = weight / (height²)

5. It displays the appropriate weight category based on the calculated BMI.

##  Example

```

What is your weight in kilograms? 68

```
