Tip Calculator

Project Description
The Tip Calculator is a simple Python program that helps users calculate how much each person should pay when splitting a bill.

The program asks the user for:
- The total bill amount
- The percentage of tip they want to give
- The number of people splitting the bill

It then calculates the final amount each person needs to pay.

How It Works
1. The program welcomes the user.
2. It asks for the total bill amount.
3. It asks for the desired tip percentage.
4. It asks how many people are splitting the bill.
5. It calculates the total bill including tip.
6. It divides the total amount between everyone and displays the result.

## Example


Welcome to the tip calculator.

What was the total bill? $100
What percentage tip would you like to give? 10
How many people to split the bill? 5

Each person should pay: $22.00
