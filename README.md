## 📚 Topics Covered

Throughout this challenge, I'll explore:

- **Python Fundamentals:** Variables, data types, control flow, functions 
- **Intermediate Python:** List/dictionary comprehension, error handling, file handling 
- **Object-Oriented Programming (OOP):** Classes, objects, methods, inheritance 
- **GUI Development:** Tkinter for desktop applications 
- **Web Development:** Flask, HTML, CSS, Bootstrap 
- **Data Science:** Pandas, NumPy, Matplotlib, Scikit-Learn 
- **Automation:** Selenium, BeautifulSoup, Requests 
- **APIs & Database:** REST APIs, SQL, JSON 
- **Game Development:** Turtle, PyGame

## 🔧 Getting Started

### Clone the repository:
```bash
git clone https://github.com/fikerbelay/100-days-of-python-100-projects.git
cd 100-days-of-python-100-projects
```

### Install dependencies:
```bash
pip install -r requirements.txt
```

### Run a specific project:
```bash
python DayXX/project-name/main.py
```
