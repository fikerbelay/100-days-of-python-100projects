Band Name Generator

Project Description
The Band Name Generator is a simple Python program that creates a unique band name based on user input.

The program asks the user for:
- The name of the city they grew up in
- Their pet's name

It then combines these two answers to generate a possible band name.

## How It Works
1. The program welcomes the user.
2. It asks for the user's childhood city.
3. It asks for the user's pet's name.
4. It combines both answers and displays the generated band name.

## Example

Welcome to the Band Name Generator.
What's the name of the city you grew up in? London
What's your pet's name? Rocky

Your band name could be London Rocky
